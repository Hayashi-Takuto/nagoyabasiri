# Claude Code 設定

## 言語設定

**すべての応答は日本語で行ってください。**

- コードのコメントも日本語で書く
- エラーメッセージの説明も日本語
- 技術用語は必要に応じて英語のまま使用可（例: API, React, TypeScript）
- コミットメッセージは日本語で提案

## コミュニケーションスタイル

- 簡潔でわかりやすい説明を心がける
- 専門用語は噛み砕いて説明
- 必要に応じて例を示す
- 敬語を使用（ですます調）

## コード生成

- コード内のコメントは日本語
- 変数名・関数名は英語（キャメルケース）
- エラーメッセージは日本語
- ログ出力は日本語

## 例

```typescript
// ユーザー情報を取得する
async function getUser(userId: string) {
  // データベースからユーザーを検索
  const user = await db.users.findUnique({ where: { id: userId } })
  
  if (!user) {
    throw new Error('ユーザーが見つかりません')
  }
  
  return user
}
```

## Git

- コミットメッセージは日本語
- 例: `feat: ユーザー認証機能を追加`
- 例: `fix: ログイン時のバリデーションエラーを修正`
- 例: `docs: READMEを更新`
